## 🎯 Quick Start Guide

### Step 1: Upload Documents
1. Go to: http://localhost:3001/documents
2. Drag & drop a .txt or .pdf file
3. Click "Upload"
4. Wait for success message

### Step 2: Start Chatting
1. Go to: http://localhost:3001/chat
2. You'll see the new ChatGPT-style interface:
   - Beautiful welcome screen with gradient icon
   - "How can I help you today?" message
3. Type your question in the input box
4. Press Enter to send
5. Watch the AI response stream in with the green "AI" avatar

## ✨ New ChatGPT-Style Features

### What You'll See:
- **Full-width messages** (not bubbles)
- **User messages**: White background with blue "U" avatar
- **AI messages**: Gray background with green "AI" avatar
- **Auto-resizing input**: Grows as you type
- **Send button**: Arrow icon inside the input box
- **New Chat button**: Start fresh conversations
- **Typing indicator**: Animated dots with AI avatar

### Try These Features:
1. Type a long message - watch the input grow
2. Press Shift+Enter for new lines
3. Click "New Chat" to start over
4. See the smooth streaming responses
5. Notice the alternating message backgrounds
